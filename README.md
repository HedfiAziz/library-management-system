# 📚 Library Management System (C)

## 🚀 Overview
A console-based library management system developed in C.  
It simulates a real-world system for managing books, loans, and statistics.

This project was built as part of my learning journey in data structures and C programming.

---

## ⚙️ Features

### 📖 Book Management
- Add / Delete / Update books
- Search books by title
- Display available books

### 📦 Loan System
- Borrow books with validation
- Return books
- Track active loans
- Check overdue books

### 📊 Statistics
- Most borrowed books
- Never borrowed books
- Top borrowers
- General library statistics

---

## 🧠 Concepts Used
- Arrays (1D & 2D)
- String manipulation
- Input validation
- Menu-driven program design
- Basic data modeling
